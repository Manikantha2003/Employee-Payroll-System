Employee Payroll System

Description
Employee Payroll System is a backend Java application developed using Object-Oriented Programming 
concepts to manage employee records and salary processing.

Features
- Add Employee
- Update Employee Details
- Salary Calculation
- Payroll Processing
- Object-Oriented Design

Technologies Used
- Java
- OOP Concepts
- Collections Framework
- Git & GitHub

Concepts Used
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
